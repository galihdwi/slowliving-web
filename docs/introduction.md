# Admin Iuran RT API

Dokumentasi ini berisi kontrak API untuk aplikasi administrasi RT.

API digunakan untuk mengelola:

- autentikasi admin
- data penghuni
- data rumah dan histori hunian
- invoice iuran bulanan
- pembayaran iuran
- pengeluaran
- laporan ringkasan dan detail bulanan

Base URL local:

```txt
http://127.0.0.1:8000/api
```

Untuk endpoint selain login, gunakan token Bearer dari endpoint login.
